Website Design. Report.

WebDev Report

The website is a fully responsive website with a simplistic approach to its design. Its easy to navigate from one page to another with a simplistic navigation approach.

Bootstrap. The website employs wholly bootsrap in the design backbone. The responsive nature of the pages is due to the the bootstrap css.

The webpages have a common foundation format:

1.Menu display. The menu has navigation links to the various pages present in the website such as the home page link. Tha navigation section also incorporates a dropdown nav-link that directs the users of the website to the products area on the website. This products pages includes one to a page displaying smartphone products as well as another to a tv page. In total there are six webpage links with three being displayed within the dropdown link. Within the menu bar there is also a search bar that is meant to direct the user to a product on the website, although its functionality is dormant. Also within the same menu bar there is a user icon which upon being clicked should direct the usre to login in to their account on the website so as to purchase items. The button functionality work displayin a pop up window throught the use of bootstrap components requesting one to fill a form. When the webpage is reduced to a small media device screen size the menu changes to a drop down navigation displayed as button with a bar icon

2.Logo. The logo is simple and to the point. It displays the website name with some font styling used to create an appeal. The logo links to the main page which is the home page.

3.Content Section. This section falls below the logo section. All the content being advertised and sold falls in this section.

4. Footer. The footer employs a bootstrap theme. Its two section combined. One part has content and the other is the copyright section. The content are displays the website logo, navigation links to webpages within the website, a news section and a social media section to link the websites social media pages. The logo links to the home page. The social media links connect to main pages of the social media pages

Index Page. The content section features a slidesjow of whats new on the website and the trendiest products. Beneath the slideshow is a card section that shows the main products sold on the website with button linking to the respective pages. There is a banner section that shows sales and discounts on the website.

Products page. The products page combines three pages with the main products being sold which include smart-tvs, smartphones and monitors. Upon accessing the products pages the content backbone format is the same. There is the display section for the products with an add to cart button. Th purpose of the add to cart is to add the product to the users basket linked to their account. Its functionality is dormant. At the bottom of each page is is a pagination from bootstrap components that links the three products pages together offers another means of navigating the products pages.

Contact Us Page. The content section displays the the companys contact details. This includes customer service contact details. Included is a contacts frm that requires user to fill in details in the event they decide to contact the company

About
